# python-zero
Esqueleto zerado de um projeto Python3 mínimo

## Para iniciar o desenvolvimento

Se você usa Linux ou outro Unix:
```sh
$ ./venv.sh
# O script lhe dará instruções para entrar no ambiente virtual
$ pip install -r requirements.txt
```

Se você usa Windows:
```
$ venv.cmd
# O script lhe dará instruções para entrar no ambiente virtual
$ pip install -r requirements.txt
```
